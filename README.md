🌦️ SkyCast - Weather App

SkyCast is a simple and responsive weather application built using JavaScript. It fetches real-time weather data using APIs and displays it in a clean UI.

🚀 Features

- Search weather by city name
- Real-time weather data
- Temperature, humidity, and weather conditions
- Dynamic weather icons
- Responsive design for all devices

🛠️ Tech Stack

- HTML
- CSS
- JavaScript (Async/Await, Fetch API)

📚 Concepts Used

- Asynchronous JavaScript (Async/Await)
- API handling
- DOM manipulation
- Error handling

🎯 Future Improvements

- Add 7-day forecast
- Add location-based weather

💻 How to Run

1. Clone the repository
2. Open "index.html" in your browser
